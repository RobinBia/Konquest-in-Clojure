# Konquest

In diesem Projekt wird ein leicht modifizierter Konquest-Klon in Clojure erstellt.

## Spiel starten

- Eclipse starten
- konquest.gui in REPL laden.
- (parameter-auswahl) ausführen - Steht in der letzten Zeile von "konquest.gui"
